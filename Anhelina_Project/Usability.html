Website Usability

The main rule of Usability: Don`t make me think.

If you find an element of your interface requires instructions, then you need to redesign it

—  Dan Rubin

Faced with the fact that your users are whizling by, there some important things you can do to make sure they see and understand as much of what they need to know - and what you want them to know - as possible:

  • Take advantages of conventions
  • Create effective visual hierarhies
  • Break pages into clearly defined areas
  • Make it obvious what is clikable
  • Eliminate distractions
  • Format content to support scanning

Using contrasting colors will help users to understand and differentiate the content you have designed. If you are having trouble deciding on an accessible color palette you can visit Color Safe which creates accessible color palettes for your site or app based on Web Content Accessibility Guidelines (WCAG).

Whitespace is also an important design aspect because it provides natural breaks and allows the user to distinguish what goes together.

A good way to provide visual breaks on a website is by using horizontal lines.

Usability Heuristics

  • Visability: the Website visibility is supported by highlighting current link on the navigation panel.

    • It does not matter how many times I have to click, as long as each click is mindless, unabiguous choice

  • Match between system and the real world: the buttons clearly correspond to the labelling . User knows exactly where he/she is going.
  • User control and freedom: control is supported through navigation and button back which bring him exactly to the previous link.
  • Consistency and Standards: fonts and link format are unified through all the the Website, all pages in this website share the same navigation bar, footer and hero-part.
  • Error privantion: validators links show mistakes of the code.
  • Recognition rather than recall: the amount of information on every page is balanced, all links have the same format.
  • Aesthetic and minimalist design: the design is clear and Aesthetic. Colour theme, small pictures and aesthetic elements are used in a decent way.

    • Get rid of half the words on each page, then get rid of half of what`s left.

  • Flexibility and Efficiency of use:when user click the link to external site the Webpage will be opened in the new tab and it is not the case if user is navigating on current The Website.
  • Help users recognize, diagnose, and recover from errors: the navigation bar and back buttons help users to deal with errors.
  • Help and documentation: page "About" the provides information about website.
